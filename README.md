# 🏠 House Price Prediction using Machine Learning

## 📌 Overview
This project predicts house prices using Machine Learning regression models. It includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, evaluation, and prediction.

---

## 🎯 Objective
- Predict house prices using regression techniques.
- Perform preprocessing and feature engineering.
- Compare multiple regression models.
- Evaluate model performance.

---

## 📂 Dataset
**Source:** https://www.kaggle.com/datasets/bhanupratapbiswas/house-price-prediction

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Joblib
- Google Colab

---

## 🤖 Models Used
- Linear Regression
- Random Forest Regressor
- Gradient Boosting Regressor

---

## 📊 Evaluation Metrics
- MAE (Mean Absolute Error)
- RMSE (Root Mean Squared Error)
- R² Score

---

## 🚀 How to Run

1. Open the notebook in Google Colab or Jupyter Notebook.
2. Install the required libraries.
3. Run all notebook cells in order.
4. Use the prediction cell to predict house prices for new data.

## ✅ Results
- Performed EDA and feature engineering.
- Compared three regression models.
